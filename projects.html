This is my portfolio projects page. It showcases some of the awesome course projects I've completed. Keep in mind this page is still a work in progress; more projects will be added as I continue my journey.

  • Movie Seat Booking App

    Key JS highlights: uses localStorage to persist data upon refresh, runs functions to dynamically calculate cost based on seat cost and number of seats selected, and uses event listeners to allow users to click on the seats they wish to purchase.

  • Form Validator App

    Key JS highlights: uses input validation to check if fields are populated correctly, changes field border color based on result of validation check, and displays unique messages for different input errors.

  • Exchange Rate Calculator App

    Key JS highlights: fetches data from a RESTful API (https://exchangerate.host/) in real-time to display current exchange rates, dynamically converts one currency to a different currency, and allows user to swap input/output currency using "Swap" button.

  • Expense Tracker App

    Key JS highlights: uses localStorage to persist data upon refresh, allows users to input custom transactions, and dynamically calculates balance based on inputted transactions.
